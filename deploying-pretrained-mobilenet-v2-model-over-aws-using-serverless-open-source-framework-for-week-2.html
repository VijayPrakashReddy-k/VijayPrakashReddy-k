Deploying pretrained MobileNet V2 model over AWS using serverless open source framework for week 2

By Vijayprakash
 articles
Jul 30, 2020
  • 0
  • Pretrained MobileNet V2 model is deployed on AWS s3 bucket
  • Serverless open source framework is used to build and run application on AWS.
  • Serverless framwork manages all the resources in AWS and user need to just focus on their Application and problem solving.
  • All the AWS resources such as API end point, Lambda functions, Cloud Formations, application packages on S3 and many mores resources are created automatically.
  • It's very cool as it takes all the burden of managing AWS resources from the user.
  • Application is deploy as AWS Lambda function which fetch model from S3 bucket.
